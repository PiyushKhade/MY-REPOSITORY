App Name: XPWithin

Tech Stack:
- Frontend: React Native
- Backend: Node.js + Express
- Database: MongoDB + Mongoose

This app is a discipline-driven fitness journey platform where users level up in real life by completing structured daily challenges.
It is designed around one simple idea:
Growth is earned daily, not rushed, not skipped, not cheated.

The app turns fitness into a structured progression system, similar to leveling up in a game, but based on real effort.

- Each user progresses through levels.
- Each level contains a fixed number of daily challenges.
- Only one day can be completed per calendar day.
- Completing all tasks for the day advances progress.
- Missing a day resets streak (discipline matters).
- Progress is visible and measurable.
